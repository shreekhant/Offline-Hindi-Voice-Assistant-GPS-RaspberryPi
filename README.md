# Hindi Offline Voice Assistant with GPS

## Features
- Offline Hindi speech recognition (Vosk)
- Offline Hindi TTS (Piper)
- GPS based city & state detection
- Runs fully offline on Raspberry Pi

## Hardware
- Raspberry Pi 4
- USB Microphone
- 3.5mm Speaker
- NEO-6M GPS module

## Required Models (Download manually)

### Vosk Hindi model
https://alphacephei.com/vosk/models/vosk-model-hi-0.22.zip

### Piper Hindi TTS
https://huggingface.co/rhasspy/piper-voices/tree/main/hi/hi_IN/pratham/medium

## Run

python3 hindiwithgps.py
